国内图床服务对比
发表于 2019-10-07 22:56| 更新于 2019-10-23 13:46|日常工具

前两篇文章都说到了图床：

  • PicGo-图片上传、管理新体验

  • Markdown Nice-微信公众号排版神器

那么图床到底是一个什么样的存在，存在的作用是什么？而现在国内又有很多云服务商，到底应该如何选择呢？接下来我给大家大致介绍一下。

图床是什么

图床就是专门用来存放图片，同时可以通过一个 http 链接直接访问图片的网络存储空间。

因为图片一般都比较小，而且需要直接在浏览器访问，所以一般在云服务商选择对象存储这类产品来充当图床功能。

有了图床后，就可以上传、下载、删除、浏览图片。

图床如何使用

选择一家云服务商，然后注册账号并进行实名认证（因为国内监管较严，所以每一家都会要求实名认证），再开通其对象存储产品，就可以直接在服务商的控制台界面对图片进行管理啦。

在实际使用时，为了更快的速度和更便宜的价格需要搭配 CDN 使用。

（对，没错！多一个 CDN 反而费用更便宜，这是怎么回事呢？继续看后文就能明白了。）

既然如此，那么剩下的问题就是怎么选择服务商了。

注意事项

  1. 腾讯虽然有免费额度，但是从2019年1月1日后开通的用户只有6个月50GB存储空间免费，无免费流量，所以相当于没有免费额度了。
  2. 阿里云只有 5GB 免费存储额度，其他无免费额度，而且必须是香港地区，香港地区的单价比国内又贵一些，所以基本上算是无免费。
  3. 七牛云需要同时开启 CDN ，才能实现免流，而免费流量不支持 HTTPS 请求。
  4. 又拍云的免费额度是在网站放又拍云的 logo 和链接，而且是以优惠券的形式发放到账户，优惠券有效期6个月。

又拍云的存储价格比看上去稍便宜一些，计算规则为：（当日流量 - 当日存储占用空间）* 存储单价，也就是说每天的流量大于或等于存储占用空间就可以免费存储。

注意啦：虽然又拍云看起来如此优惠，可以免费存储，但是实际算下来和别家价格一样，因为他家的 CDN 流量价格相当于别家的存储价格 + CDN 价格，不信往下看。

如何选

免费额度足够如何选择？

优选网易云，但是不开 CDN；然后是 ucloud。

  1. 网易云送的最多。
  2. ucloud 送的流量相对少，而且 ucloud 的 CDN 贵不少，所以转收费也不转它。
  3. 七牛云免费流量不支持 HTTPS 请求。
  4. 又拍云的免费额度是在网站放又拍云的 logo 和链接，而且是以优惠券的形式发放到账户，优惠券有效期6个月。

免费额度不够用如何选择？

网易云、网易云、阿里云三家随便选一个你方便或顺眼的。

  1. 网易云、腾讯云、阿里云的对象存储下行流量很贵，但是配合 CDN 服务其实就比其他几家都便宜。
  2. 对于量小的用户，目前网易云价格最优惠，而且有免费额度，虽然网易云存储单价比腾讯和阿里多几分钱，但是1GB 空间一个月才多几分钱，100GB 一个月也才多2元钱多一点，所以流量的价格更敏感一些。
  3. 量大的用户其实选择性不多，有很多实际条件或应用场景限制，而且对价格其实不太敏感，加上可以议价，所以随便选吧。

总之应证了那句话：规模越大，费率越低。
